How to setup ant with Retroduction Carma
----------------------------------------

Ant needs to be enhanced by the carma artifacts and carma's runtime dependencies by:

- Adding all Carma artifacts to either ~/.ant/lib or $ANT_HOME/lib
- Adding libraries bundled in the Carma Ant Plugin assembly's ./lib/ folder to either
  ~/.ant/lib or $ANT_HOME/lib

Add the following snippets to the build.xml file of the project under test

Carma's Core Engine Integration (End user variant):

	<taskdef name="carma" classname="com.retroduction.carma.ant.AntCarmaTest" />

	<target name="carma">
		<carma classesDir="<PATH TO PROJECT UNDER TEST'S CLASS DIRECTORY>" testClassesDir="<PATH TO PROJECT UNDER TEST'S TEST-CLASS DIRECTORY>" dependencyClassPathUrls="??????"/>
	</target>
